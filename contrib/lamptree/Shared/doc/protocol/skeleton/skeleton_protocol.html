2. Introduction

The perfSONAR software suite is an infrastructure for network performance monitoring, making it easier to solve end-to-end performance problems on paths crossing several networks. This set of independent software services is focused on the task of delivering performance measurements within a federated environment. These services act as an intermediate layer, between the functions provided by the performance measurement tools and that of the diagnostic and visualization applications (i.e network weathermaps, [perfSONAR-UI], etc.). The goal of this layer is making and exchanging performance measurements between networks, utilizing well-defined protocols.

Terms

C

characteristic

Taken on the context of networking, these describe the intrinsic properties of a portion of the network that are related to the performance and reliability of the network. See http://www.ggf.org/documents/GFD.23.pdf for more information.

N

NM-WG

The performance of most grid applications is dependent on the performance of the networks forming the grid. The Network Measurements Working Group (NMWG) identifies network metrics (aka characteristics) useful to grid applications and middleware, and develops standard mechanisms to describe and publish these characteristics to the Grid.

S

schema

XML specification, normally written in XML.

schemata

Plural of schema.
